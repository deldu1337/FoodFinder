### PART 3. 화면 구성 및 세부 기능
 #### 메인 화면
 1. 우측 상단 로그인 버튼 클릭 시 로그인 화면 이동
 2. 상단에 앱 로고 표시
 3. 레시피, 식당 추천, 냉장고, 게시판 기능이 구현되어 있으며 각각의 기능은 썸네일이 존재
 4. 구현된 기능 중 하나를 클릭할 시 해당 기능 화면 이동
